Analysis¶

So, what do the above plots show us?

Observation: full model separates the test data even better than the training data.
Why: the test data barcode is 28% longer that the training data barcode.
Implications: the model is robust, however it contains uninformative features.

Observation: pruned model separates the test data and the training data similarly well.
Why: the training data barcode is 2% longer that the test data barcode.
Implications: the model is almost as robust as the full model and it does not contain uninformative features.
